# teacher-directory

## Installation:

1. Clone the repository

2. Install pip virtual environment and dependencies
  
  ```python
  pipenv install
  
  # initialize virtual environment
  pipenv shell
  
  # once inside,
  pip install -r requirements.txt
  ```
  
3. Migrate
  ```python
  python manage.py migrate
  ```
  
4. Run server
  ```python
  python manage.py runserver
  ```
